# 我家厨房 - 微信点菜小程序

一个基于微信云开发的点菜小程序，支持厨师和食客协作管理日常饮食。

## 功能特性

- **厨房管理** - 1位厨师 + 最多3位食客
- **菜品管理** - 分类管理、规格设置、图片上传
- **购物车** - 添加菜品、选择规格、数量调整
- **订单管理** - 下单、状态更新、历史记录
- **美食日历** - 每日美食照片打卡
- **主题切换** - 粉色、白色、绿色三种风格
- **邀请功能** - 通过厨房ID邀请家人加入
- **订单通知** - 厨师端轮询检测新订单（需小程序前台运行）

## 技术栈

- 微信小程序（WXML/WXSS/JS）
- 微信云开发
  - 云函数
  - 云数据库
  - 云存储

## 使用说明

1. 克隆项目到本地
2. 使用微信开发者工具打开
3. 配置云开发环境
4. 部署云函数
5. 编译运行

## 角色说明

| 角色 | 权限 |
|------|------|
| 厨师 | 管理菜品、管理订单、上传照片 |
| 食客 | 浏览菜品、下单、上传照片 |
